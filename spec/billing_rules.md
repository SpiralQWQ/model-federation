# Token 联邦计费播报规范

> **触发方式**：每次模型调用后自动触发（无需说"计费/账单"）
> **用途**：汇总 Token 消耗和费用，支持写入协同日志/程序员日志头部

## 计费类型标记

| 类型 | 标记 | 适用模型 |
|------|------|---------|
| 按量扣费 | ⚠️ | qwen3.7-max, glm-4.7 等 |
| Google 付费 | 💳 | gemini-3.5-flash |
| 赠送额度内 | ℹ️ | qwen-long, glm-4.5-air 等 |
| 免费模型 | ✅ | GLM-4.6V-Flash |
| 积分支付 | 💎 | suitui-ai 所有模型 |

## 播报格式

```
📊 [联邦计费] DeepSeek(0) + Qwen(qwen-max ⚠️ 500→200) + GLM(glm-4.5-air ℹ️ 100→50) = 总计 850 Token
```

## 会话 Token 累计表（写入日志头部用）

每次模型调用后自动累计，写入日志时从下表汇总填入"Token 消耗"和"花费反馈"字段。

```
📊 [会话 Token 累计]
   引擎       模型                   输入→输出 Token   计费类型
   ─────────────────────────────────────────────────────────
   DeepSeek   claude-sonnet-5        12,345→2,100      💎 订阅（不计）
   Qwen       qwen3.7-plus           3,200→800        ⚠️ 按量扣费
   GLM        glm-4.5-air              800→200        ℹ️ 赠送额度
   ─────────────────────────────────────────────────────────
   总计                              16,345→3,100     = 19,445 Token
   模型涉及: claude-sonnet-5 / qwen3.7-plus / glm-4.5-air
```

**填入日志格式**：
- `Token 消耗: 19,445 Token（claude-sonnet-5 + qwen3.7-plus + glm-4.5-air）`
- `花费反馈: Qwen ⚠️ 按量扣费 / GLM ℹ️ 赠送额度`

## 降级兜底

若任何一方 API 报错：
1. 打印警告：`⚠️ [联邦降级] <模型名> 不可用: <原因>`
2. 告知用户哪个引擎缺失
3. 询问是否重新连接
